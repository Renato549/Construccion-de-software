Preguntas a resolver

¿Cuál es la diferencia entre Internet y la World Wide Web?

La diferencia se establece que el internet como una gran red de computadoras alrededor de todo el mundo conectadas entre sí, por otra parte la web o la establecida como World Wide Web es una gran colección de páginas que se asienta en la red antes ya mencionada.



¿Cuáles son las partes de un URL?

Las partes se conocen como:

  • Protocolo: la mayoría de las veces son HTTP (o HTTPS para una versión segura de HTTP).
  • Dominio: nombre que se utiliza para identificar una o más direcciones IP donde se encuentra el recurso.
  • Ruta: especifica la ubicación del recurso en el servidor.
  • Parámetros: datos adicionales utilizados para identificar o filtrar el recurso en el servidor.


¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

Sirven para indicar una accion sobre un recurso en concreto.

Sobre el ciclo de vida y desarrollo de los sistemas de información:

¿Cuál es el ciclo de vida de los sistemas de información?

  • Fase de planificación donde se prepara el diseño e implementación del sistema. Aquí se define el alcance del proyecto, justificacion y metodología.
  • Fase de análisis donde se busca realizar requisitos que guiaran el sistema y sus cambios si es que se busca una actualizacion.
  • Fase de diseño, se determina cómo el nuevo sistema de información cumple con los requisitos establecidos.
  • Fase de desarrollo, se inicia la produccion del proyecto.
  • Fase de integración; En esta etapa es asegurar que el diseño ideado cumple con los requisitos.
  • Fase de implementación; Aqui se hace la instalación y actualizacion del hardware y/o software diseñado.



Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

El uso de !important es para dar prioridad a instrucciones, de manera que las siguientes indicaciones no puedan sobrescribirla. Se considera una mala práctica ya que dificulta la depuracion del codigo porque rompre la cascada natural de las hojas de estilo.



Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Por que el fondo puede arruinarse debido a las caracteristicas de la imagen.



Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

Lo ideal es usar px o pixeles, debido a que es la medida con la que se trabaja para una interfaz.



¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Por el “tamaño” de la página. Si se desea que los usuarios tengan un rapido acceso a los contenidos de nuestro proyecto, debemos hacer que los elementos incluidos en el diseño no pesen demasiado.
